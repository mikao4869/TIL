# 정의 
 react-select는 **React에서 커스텀 드롭다운(select 박스)**을 만들기 위한 강력한 라이브러리야. 기본 <select>는 브라우저의 네이티브 UI라서 커스터마이징이 거의 불가능한데, react-select는 그걸 완전히 내 마음대로 꾸밀 수 있게 해줘.

## 설치


      npm install react-select``
